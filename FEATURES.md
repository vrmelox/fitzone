# FONCTIONNALITÉS DE LA PLATEFORME GOGYM

## Vue d'ensemble
Gogym est une plateforme de gestion de salle de sport avec trois types d'utilisateurs : Membres (Utilisateurs), Réceptionnistes et Administrateurs.

---

## 👤 UTILISATEUR (MEMBRE)

### Dashboard Principal
- Visualiser les statistiques personnelles (visites, temps d'entraînement, calories)
- Voir la progression mensuelle
- Accéder aux cours à venir
- Consulter les insights personnalisés (créneaux favoris, zones préférées)

### Planning & Cours
- Consulter le planning hebdomadaire des cours
- Réserver des cours collectifs
- Voir les places disponibles
- Filtrer les cours par catégorie/niveau
- Gérer ses réservations

### Statistiques
- Visualiser les check-ins du mois
- Suivre le temps d'entraînement moyen
- Consulter les calories brûlées
- Voir les objectifs atteints
- Analyser la répartition des activités (Musculation, Cardio, HIIT)
- Suivre la progression mensuelle

### Profil
- Gérer les informations personnelles (nom, email, téléphone, adresse)
- Définir les objectifs fitness
- Consulter les informations de santé
- Modifier la photo de profil

### Abonnement & Paiements
- Consulter le type d'abonnement actuel
- Voir la date d'expiration
- Accéder à l'historique de facturation
- Télécharger les factures
- Gérer les moyens de paiement

### Plan de la Salle
- Visualiser le plan interactif de la salle
- Voir l'occupation en temps réel par zone
- Consulter les horaires d'affluence
- Planifier ses séances selon la fréquentation

### Paramètres
- Gérer les notifications (email, push, SMS)
- Modifier l'apparence (thème)
- Changer la langue
- Accéder à l'aide et support
- Se déconnecter

## 🔐 SÉCURITÉ & PERMISSIONS

### Utilisateur (Membre)
- Accès uniquement à ses propres données
- Pas d'accès aux données des autres membres
- Pas d'accès aux fonctions administratives

### Réceptionniste
- Accès en lecture/écriture aux données membres
- Gestion des check-ins et réservations
- Pas d'accès aux finances détaillées
- Pas de modification des paramètres système

### Administrateur
- Accès complet à toutes les fonctionnalités
- Gestion de tous les utilisateurs
- Accès aux données financières
- Configuration système complète
- Gestion des permissions
